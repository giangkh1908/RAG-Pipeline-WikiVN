## Kiến trúc

```
records (generator)
  │
  │  for batch in batches(records, 500):
  │
  ├── Stage 1: ProcessPoolExecutor ────────────────────────
  │   Worker: normalize → clean → split_paragraphs
  │   → file: pipelines/workers.py (chỉ import paragraph utils + models)
  │   → file: transform/paragraphs.py (chỉ re, không ML)
  │   → worker KHÔNG import chunker.py hay ingest_pipeline.py
  │
  ├── Stage 2: Main process ───────────────────────────────
  │   Dedup → model.encode(all_paragraphs, batch_size=...)
  │   Per-doc: merge_by_similarity → split_if_large → chunks
  │
  ├── Stage 3: ThreadPool (ParallelEmbedder) ──────────────
  │   _flush_batch: sub-batch → OpenRouter song song
  │   → thread-local httpx.Client, rate-limit retry
  │
  └── Stage 4: Accumulate (doc_id, updated) ──────────────
      → list[tuple[str, bool]], giải phóng batch data
```

**Mỗi batch:** xử lý xong → flush → GC → batch kế. RAM tại mọi thời điểm ~200-800 MB.

"""Text splitting utilities. Zero ML dependencies — safe for worker processes."""
from __future__ import annotations

import re

_PARAGRAPH_RE = re.compile(r"\n\s*\n", re.MULTILINE)


def split_paragraphs(content: str) -> list[str]:
    """Split text into paragraphs by blank lines. Pure regex, no model."""
    return [p.strip() for p in _PARAGRAPH_RE.split(content) if p.strip()]
```

Worker import file này chỉ kéo `re` (stdlib) — 0 ML overhead.

"""Worker functions for multiprocessing. Minimal imports — no ML deps."""
from __future__ import annotations

from rag_pipeline.transform.paragraphs import split_paragraphs


def process_records_chunk(args: tuple) -> list[tuple]:
    """Worker: normalize → clean → split_paragraphs.
    
    Args: (normalizer, cleaner, records)
    Returns: list[(CanonicalDocument, paragraphs: list[str])]
    
    KHÔNG import chunker.py, sentence_transformers hay torch.
    Chỉ dùng split_paragraphs() từ transform/paragraphs.py (re thuần).
    """
    normalizer, cleaner, records = args
    output = []
    for record in records:
        doc = normalizer.normalize(record)
        doc.content = cleaner.clean(doc.content)
        paragraphs = split_paragraphs(doc.content)
        output.append((doc, paragraphs))
    return output
```

def encode_paragraphs(self, paragraphs: list[str]) -> np.ndarray:
    """Encapsulate model.encode(). Gọi từ main process, không từ worker."""
    return self.model.encode(paragraphs, show_progress_bar=False)


def paragraphs_to_chunks(
    self,
    document: CanonicalDocument,
    paragraphs: list[str],
    embeddings: np.ndarray,
) -> list[DocumentChunk]:
    """Tạo chunks từ paragraphs đã encode sẵn."""
    merged = self._merge_by_similarity(paragraphs, embeddings)
    chunks: list[DocumentChunk] = []
    for group_idx, text in enumerate(merged):
        for offset, sub_text in enumerate(self._split_if_large(text)):
            chunks.append(self._make_chunk(document, sub_text, group_idx, offset, chunks))
    self._link_neighbors(chunks)
    return chunks

### 4. `indexing/embedder.py` — ParallelEmbedder + fix

```python
@dataclass(slots=True)
class ParallelEmbedder:
    inner: Embedder
    max_workers: int = 4
    sub_batch_size: int = 64

    def embed_texts(self, texts: list[str]) -> list[list[float]]:
        sub_batches = [
            texts[i:i + self.sub_batch_size]
            for i in range(0, len(texts), self.sub_batch_size)
        ]
        if len(sub_batches) <= 1:
            return self.inner.embed_texts(texts)

        from concurrent.futures import ThreadPoolExecutor, as_completed
        with ThreadPoolExecutor(max_workers=self.max_workers) as pool:
            futures = {pool.submit(self.inner.embed_texts, sb): i
                       for i, sb in enumerate(sub_batches)}
            results = [None] * len(sub_batches)
            for f in as_completed(futures):
                idx = futures[f]
                results[idx] = f.result()
        return [v for r in results for v in r]
```

@dataclass(slots=True)
class OpenRouterEmbeddingClient:
    config: EmbeddingConfig
    _thread_local: threading.local = field(
        default_factory=threading.local, init=False, repr=False
    )

    def _get_client(self) -> httpx.Client:
        if not hasattr(self._thread_local, "client"):
            self._thread_local.client = httpx.Client(
                base_url=self.config.api_base,
                timeout=self.config.timeout_seconds,
                limits=httpx.Limits(max_keepalive_connections=8, max_connections=16),
            )
        return self._thread_local.client

    def embed_texts(self, texts: list[str]) -> list[list[float]]:
        api_key = os.getenv(self.config.api_key_env)
        if not api_key:
            raise RuntimeError(...)
        payload = {"model": self.config.model_name, "input": texts}
        headers = {"Authorization": f"Bearer {api_key}"}
        client = self._get_client()
        for attempt in range(3):
            try:
                response = client.post("/embeddings", json=payload, headers=headers)
                response.raise_for_status()
                data = response.json()["data"]
                return [item["embedding"] for item in data]
            except httpx.HTTPStatusError as e:
                if e.response.status_code == 429 and attempt < 2:
                    import time
                    time.sleep(2 ** attempt)
                    continue
                raise

### 5. `pipelines/ingest_pipeline.py` — thay đổi chính

#### Sửa `run()` — return `list[tuple[str, bool]]`, xoá `boundaries` dead code

```python
def run(self, records: Iterable[SourceRecord]) -> list[tuple[str, bool]]:
    """Sequential. Returns list of (doc_id, updated)."""
    results: list[tuple[str, bool]] = []
    seen: set[str] = set()
    pending: list[tuple] = []

    for record in records:
        document = self.normalizer.normalize(record)

        if document.checksum in seen:
            results.append((document.doc_id, False))
            continue
        if self.vector_store.has_document_version(document.doc_id, document.checksum):
            seen.add(document.checksum)
            results.append((document.doc_id, False))
            continue

        seen.add(document.checksum)
        document.content = self.cleaner.clean(document.content)
        chunks = self.chunker.chunk(document)
        pending.append((document, chunks))

        if self._pending_chunks(pending) >= self.embed_batch_size:
            self._flush_batch(pending, results)

    if pending:
        self._flush_batch(pending, results)

    return results
```

#### Sửa `_flush_batch()` — xoá `boundaries`, append `(doc_id, True)`

```python
def _flush_batch(self, pending: list[tuple], results: list[tuple[str, bool]]) -> None:
    all_texts: list[str] = []
    for _, chunks in pending:
        all_texts.extend(chunk.text for chunk in chunks)

    vectors = self.embedder.embed_texts(all_texts)

    pos = 0
    for document, chunks in pending:
        n = len(chunks)
        doc_vectors = vectors[pos : pos + n]
        indexed_chunks = [
            IndexedChunk(chunk=chunk, dense_vector=vec, sparse_vector=None)
            for chunk, vec in zip(chunks, doc_vectors)
        ]
        self.vector_store.upsert_document(document.doc_id, document.checksum, indexed_chunks)
        results.append((document.doc_id, True))
        pos += n

    pending.clear()
```

#### Thêm `run_parallel()` — streaming batch

```python
from rag_pipeline.pipelines.workers import process_records_chunk


def run_parallel(
    self,
    records: Iterable[SourceRecord],
    processing_workers: int = 2,
    doc_batch_size: int = 500,
) -> list[tuple[str, bool]]:
    """Streaming parallel pipeline."""
    import itertools
    from concurrent.futures import ProcessPoolExecutor

    all_results: list[tuple[str, bool]] = []
    seen: set[str] = set()

    with ProcessPoolExecutor(max_workers=processing_workers) as pool:
        records_iter = iter(records)
        while True:
            batch = list(itertools.islice(records_iter, doc_batch_size))
            if not batch:
                break

            # ── Stage 1: Multiprocessing ──
            groups = _split_records(batch, processing_workers)
            worker_args = [
                (self.normalizer, self.cleaner, group)
                for group in groups
            ]
            stage1_out: list[tuple] = []
            for chunk_result in pool.map(process_records_chunk, worker_args):
                stage1_out.extend(chunk_result)

            # ── Stage 2a: Dedup ──
            doc_entries: list[tuple] = []   # (doc, start_idx, n_paragraphs)
            all_paragraphs: list[str] = []

            for doc, paragraphs in stage1_out:
                if doc.checksum in seen:
                    all_results.append((doc.doc_id, False))
                    continue
                if self.vector_store.has_document_version(doc.doc_id, doc.checksum):
                    seen.add(doc.checksum)
                    all_results.append((doc.doc_id, False))
                    continue
                seen.add(doc.checksum)

                if not paragraphs:  # doc rỗng sau split → skip
                    all_results.append((doc.doc_id, False))
                    continue

                doc_entries.append((doc, len(all_paragraphs), len(paragraphs)))
                all_paragraphs.extend(paragraphs)

            if not all_paragraphs:
                continue  # batch toàn bộ trùng hoặc rỗng

            # ── Stage 2b: Single encode + chunk assembly ──
            embeddings = self.chunker.encode_paragraphs(all_paragraphs)

            pending: list[tuple] = []
            for doc, start, count in doc_entries:
                dp = all_paragraphs[start:start + count]
                de = embeddings[start:start + count]
                chunks = self.chunker.paragraphs_to_chunks(doc, dp, de)
                pending.append((doc, chunks))

                if self._pending_chunks(pending) >= self.embed_batch_size:
                    self._flush_batch(pending, all_results)

            if pending:
                self._flush_batch(pending, all_results)

    return all_results
```

#### Module-level utility

```python
def _split_records(records: list, n: int) -> list[list]:
    """Chia records thành n group gần bằng nhau."""
    avg = len(records) // n
    remainder = len(records) % n
    groups, idx = [], 0
    for i in range(n):
        size = avg + (1 if i < remainder else 0)
        groups.append(records[idx:idx + size])
        idx += size
    return groups
```

## Memory budget per batch (500 docs)

| Thành phần | Memory |
|-----------|--------|
| Raw text (500 × ~10KB) | ~5 MB |
| Python string overhead (~25K paragraphs × 60B) | ~1.5 MB |
| `all_paragraphs` list (25K refs × 8B) | ~0.2 MB |
| Chunker embeddings numpy (25K × 384 × 4B) | ~38 MB |
| IndexedChunk objects (temporary) | ~2 MB |
| Model SentenceTransformer (shared, loaded once) | ~500 MB |
| **Tổng cộng** | **~550 MB** |

So với sequential hiện tại (`results` tích luỹ 3-6GB) → giảm ~6-10x peak memory.

1. **Pickle safety**: `pickle.dumps(self.normalizer)` và `pickle.dumps(self.cleaner)` — kích thước < 1KB, không lỗi
2. **Worker không load model**: chạy `--sample 5` + log PID — worker chỉ import `paragraphs.py` + `models.py`
3. **Backward compat**: `run()` với `parallel_doc_workers=0` → output giống bản gốc
4. **Parallel**: `run_parallel()` với `parallel_doc_workers=2` → cùng số doc_ids
5. **Memory**: Task Manager / `memory_profiler` trên `--sample 10` — peak < 1GB

## Vấn đề còn nợ (không blocking)

| Vấn đề | Mô tả | Khi nào xử lý |
|--------|-------|---------------|
| `pool.map()` fail-fast | 1 group lỗi mất cả batch | Khi cần resilience |
| `_split_records` chia theo count | Doc dài ngắn khác nhau → workers không đều tải | Khi performance không đạt |
| `has_document_version` N round trips/batch | Mỗi doc 1 request Qdrant | Khi Qdrant latency thành vấn đề |
| Prefetch batch N+1 | ProcessPool rảnh trong lúc encode/embed | Khi latency là ưu tiên |
| `seen` set lớn dần | Growth = unique docs | Khi dataset >> RAM |
